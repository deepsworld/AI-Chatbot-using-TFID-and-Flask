# AI-Chatbot-using-TFID-and-Flask
A question answering chatbot that is trained using TFID and Flask is used to create the RESTFUL API

To run it on your local machine:

1. Clone the entire repository
2. Change to the directory in which the cloned repo is stored or Open terminal from that directory.
3. Once in the directory, type: 'pip install -r requirements.txt'
4. After completing the installation of required modules, type: 'python app.py'
5. Go to your browser and type: 'localhost:5000/'
